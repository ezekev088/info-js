# Talento-Lab
#### Entrega final
Este proyecto consiste en una página web básica desarrollada como parte de un curso de Front-End. La estructura del sitio está construida con HTML semántico, utilizando etiquetas como <header>, <div>, <section> y <footer> para organizar el contenido. Además, incorpora un menú desplegable que se habilita únicamente cuando la página se visualiza en dispositivos con menos de 500 px de ancho.

En el archivo CSS se emplea Flexbox para distribuir y organizar todos los elementos de la interfaz.

En cuanto a JavaScript, desarrollé un formulario de registro que solicita los datos del usuario. Una vez ingresados, la información se almacena en Local Storage para ser utilizada posteriormente en el inicio de sesión. Si los datos coinciden correctamente, el sistema redirige al usuario a la página de inicio.

Si bien la página, el logo y los productos son ficticios, el objetivo del sistema de login es poder ofrecer a los usuarios descuentos y promociones en productos seleccionados. Para lograr esto, sería necesario integrar APIs para obtener los datos reales de los productos y aplicar, mediante JavaScript, un código que calcule un descuento del 15% sobre sus precios.
